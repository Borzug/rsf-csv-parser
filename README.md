# Rally Race Chart

Electron-приложение для визуализации и анализа результатов ралли из CSV-файла (экспорт rallysimfans.hu).
Приложение полностью сгенерено через Claude.ai (даже это readme).

## Установка и запуск

```bash
npm install
npm start
```

## Экран загрузки

При запуске отображается экран приветствия:

- **Название события** — необязательное поле. Введённый текст отображается в заголовке приложения на протяжении всей сессии.
- **Выбрать CSV-файл** — открывает диалог выбора файла. После успешной загрузки приложение переходит к экрану анализа.

## Заголовок приложения

После загрузки файла в шапке отображается:

- Название события (или «Rally Race Chart» по умолчанию).
- Статистика: количество финишировавших / всего участников и количество спецучастков.
- Кнопка **← Назад** в боковой панели — возвращает на экран загрузки.

## Переключатель вкладок

Две вкладки в верхней части рабочего экрана:

- **📈 График** — интерактивный график хода гонки.
- **🏆 Результаты** — таблица итоговых позиций.

## Вкладка «График»

### Как читать график

- **Ось X** — временная шкала гонки. Каждый спецучасток (SS) разделён на три точки: SP1, SP2 и финиш SS. Под осью — названия спецучастков.
- **Ось Y** — накопленное время прохождения в формате `М:СС.д` (при ≥ 60 мин — `Ч:ММ:СС.д`).
- Все участники стартуют из точки (0, 0) и расходятся вправо-вверх. Победитель заканчивает с наименьшим значением Y.
- **Сплошная линия** — успешное прохождение участка.
- **Пунктирная линия** — участник получил Super Rally на этом участке.
- **Круглый маркер** — точка сплита.
- **Ромбовидный маркер (◆)** — точка сплита с комментарием участника.

### Взаимодействие с графиком

**Наведение курсора** — при приближении к линии графика она выделяется, остальные приглушаются. Появляется всплывающая подсказка с данными участника в этой точке.

**Клик по линии** — «закрепляет» выбранного участника:
- В верхней части графика появляется полоса с именем участника и кнопкой закрытия (×).
- На всех точках с комментариями раскрываются постоянные подсказки с текстом комментария.
- Остальные линии становятся неинтерактивными до снятия выделения.
- Повторный клик по линии или нажатие **Escape** снимает закрепление.

**Клик по названию спецучастка** на оси X — выбирает только этот SS в фильтре спецучастков. Повторный клик восстанавливает предыдущее состояние фильтра.

**Всплывающая подсказка (тултип)** содержит:
- Название этапа.
- Имя участника и автомобиль.
- Накопленное время.
- Сумму штрафов и сервисных штрафов (красным).
- Количество Super Rally (красным).
- Комментарий участника (если есть).

Если два участника имеют одинаковое время в данной точке, тултип показывает обоих.

### Легенда (правая панель)

Отображается автоматически, когда выбраны не все участники/группы/автомобили. Показывает список активных участников с цветовым индикатором, отсортированных по убыванию времени (медленнейшие — вверху). Если список не помещается по высоте, появляется кнопка **‹** для горизонтального расширения панели.

## Боковая панель фильтров

Все группы фильтров сворачиваются и разворачиваются кликом по заголовку. В заголовке каждой группы показан счётчик выбранных/всего пунктов. Кнопки **Все** и **Никого** выделяются, когда все или ни один пункт не выбраны.

### Фильтры на вкладке «График»

**Спецучастки** — выбор участков для отображения. Снятие галочки убирает SS с графика и из подсчёта накопленного времени. При выборе подмножества отсчёт начинается от первого выбранного участка.

**Участники** — управление видимостью линий на графике. Содержит строку поиска: при вводе 2+ символов список фильтруется с задержкой 200 мс. Кнопка **×** сбрасывает поиск. При наведении на строку участника соответствующая линия на графике подсвечивается. При выборе фильтра по этапу из списка автоматически скрываются участники, не стартовавшие на этом участке.

**Группа / Класс** — скрывает участников выбранных классов. При изменении фильтра список участников обновляется автоматически.

**Автомобиль** — фильтрация по конкретной модели автомобиля.

### Фильтры на вкладке «Результаты»

**Этапы (результаты)** — выбор спецучастков для расчёта итоговых времён в таблице.

**Участники (результаты)** — выбор конкретных участников.

**Класс (результаты)** — фильтрация по классу автомобиля. Лидер и отставание рассчитываются отдельно для каждого класса.

## Вкладка «Результаты»

Таблица финальных позиций, отсортированная по возрастанию суммарного времени прохождения всех этапов (быстрейший — первый).

### Столбцы таблицы

| Столбец | Описание |
|---|---|
| **#** | Итоговое место (🥇🥈🥉 для топ-3) |
| **Участник** | Логин и реальное имя |
| **Класс** | Класс автомобиля |
| **Автомобиль** | Название автомобиля |
| **Общее время** | Суммарное время по всем выбранным этапам |
| **Штрафы** | Сумма всех штрафов и сервисных штрафов |
| **Ср. отст. лидер** | Среднее отставание от лидера за участок (с учётом SR) |
| **Ср. отст. пред.** | Среднее отставание от предыдущего места за участок (с учётом SR). Красный — отставание, зелёный — опережение |
| **Чист. отст. лидер** | Среднее отставание от лидера, только на участках без SR у обоих |
| **Чист. отст. пред.** | Среднее отставание от предыдущего, только на участках без SR у обоих |
| **Общее отст.** | Полная разница времени с лидером |
| **SR** | Общее количество Super Rally |

Столбцы **Чист. отст. лидер** и **Чист. отст. пред.** имеют подчёркивание и тултип с пояснением: сколько этапов из общего числа использовалось для расчёта.

### Поиск участника

Клик по заголовку столбца **Участник** превращает его в поле ввода. При совпадении таблица прокручивается к найденной строке, которая выделяется цветом.

### Выбор строк и применение фильтра

Строки кликабельны — при клике строка отмечается как выбранная (можно выбрать несколько). При наличии хотя бы одной выбранной строки в нижней части экрана появляется кнопка **«Применить фильтр по участникам»**. Нажатие устанавливает фильтр только на выбранных участников — таблица и все расчёты пересчитываются для этого подмножества.
